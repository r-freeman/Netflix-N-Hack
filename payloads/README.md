# Netflix 'N Hack Payloads

After launching the inject.js, a JS server is populated waiting for payloads.

On this folder you have 2 examples:

- hello_world.js : Render a text on the UI using Netflix Gibbon internals

- lapse.js (port from Y2JB): generated by `bundle_lapse.sh` includes elf_loader.js

- FTP Server: allows you to access sandbox of Netflix
